>>구성품
Pomodoro.ahk	파일 코드 (따로 내용 편집하지 않고 사용할 예정이라면 삭제해도 됨)
Pomodoro.exe	실행 파일
README.txt	취급설명서
ring.mp3	시간 종료 시 울릴 벨소리


>> 설치
1. Pomodoro.exe로 즉시 실행


>> 기능
1. Start를 눌러 25분 시작
2. Pause를 눌러 정지. 정지했을 경우, Start를 눌러 다시 시작. (Pause 버튼을 다시 누르지 않게 주의하시오... 리셋됨 - 차후 개선할지도... 아닐지도...)
3. 리셋 방법: Start 더블클릭 

+exe 파일과 동일 경로에 ring.mp3를 배치할 경우 해당 mp3가 재생됨(이름.확장자만 동일하게 맞춘다면 타 파일로 교체 가능). 파일이 존재하지 않을 시 기본 Beep음 재생
